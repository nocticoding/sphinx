# sphinx

Aplicación interactiva de preguntas y respuestas para ser controlada mediante teclado para instalaciones en museos, dentro del proyecto X-ivit.

Falta:
* Obtener las preguntas y respuestas de un json externo
* Mostrar modal con las respuestas
* Extraer a configuración parámetros como el tiempo de respuesta, asignación de botones, etc
* Pasar css a Bootstrap
* Añadir botón de HOME
* Añadir pulsación para pasar a la siguiente pregunta.
* Añadir táctil -> más adelante.
* Cambiar a ES6 e integrarlo en un react.

La idea es tener una aplicación autocontenida que pueda ejecutarse en un brightsign e interaccionar con una botonera programada medianta arduino.

Posteriormente, aunque Sphinx se mantendrá como una solución independiente, también se integrará sobre X-ivit.
